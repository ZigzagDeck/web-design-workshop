## Getting Started

1. Clone this repository:
   ```bash
   git clone <your-repo-url>
   cd web-design-workshop
   ```

2. Open any experiment folder and double-click the HTML file to view in browser

## Experiments Overview

1. **Bookstore Homepage** - Three-frame layout with navigation
2. **Login Page** - Simple login form with frames
3. **Catalogue Page** - Book catalog with details
4. **Cart Page** - Shopping cart display
5. **Registration Form** - Complete registration with various input types
6. **JS Validation (Part 1)** - Name and password validation
7. **JS Validation (Part 2)** - Email and phone validation
8. **CSS Styling** - Fonts, styles, and backgrounds
9. **CSS Links** - Link states and styling
10. **Portfolio Project** - Complete portfolio website

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## Notes

- All code is written in a beginner-friendly style
- No frameworks or libraries required
- Each experiment is self-contained
- Comments included for learning

## How to Use

Navigate to any experiment folder and open the `.html` file in your web browser. No server setup required!
